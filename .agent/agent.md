# BuildBrief Project Context

## Project Overview
BuildBrief is an AI-powered architectural tool that helps users define software specifications (Mega-Prompts) through a structured interview process.

## Design System: "Cosmos"
- **Theme**: Dark Mode (Glassmorphism + Space/Nebula aesthetics).
- **Core Colors**:
    - Background: `#0B1120` (Deep Slate)
    - Primary: `#7C3AED` (Nebula Purple)
    - Secondary: `#38BDF8` (Starlight Blue)
    - Text: `#F8FAFC` (White/Slate 50)
- **UI Components**:
    - `GlassCard`: Translucent backgrounds with blur.
    - `GlowingButton`: Neumorphic, glowing interactions.
    - `Stardust`: Animated background particles.

## Technology Stack
- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express, Google Gemini API.
- **State Management**: React Context / Hooks.

## Agent Guidelines
- **Frontend**: Prioritize "Wow" factor. Animations should be smooth (spring physics). Use `lucide-react` for icons.
- **Backend**: Keep endpoints clean (`/api/v1/...`). Use strict typing.
- **Documentation**: Keep README updated.
